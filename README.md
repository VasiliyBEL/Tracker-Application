# Tracker Application
Tracker App in ASP.NET Core MVC using SyncFusion Components + Local DB(using Microsoft SQL Service Management Studio).

Implemented:
1) CRUD Operations;
2) Category & Transaction Grid;
3) Paging and Sorting Dashboard;
4) Chart Elements Side Menu.
